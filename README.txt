Плагин к Chrome (и не только), добавляющий немного удобства к FriendFeed-у.

Страница плагина в Chrome Web Store:
https://chrome.google.com/webstore/detail/friendfeed-co/cgniebblmblnalniphdgdjalefamejop

Плагин можно использовать как юзерскрипт, актуальная версия скрипта всегда доступна по адресу:
https://github.com/davidmz/friendfeed-and-co/raw/master/ffco-sac.user.js

Для запуска юзерскриптов есть разные плагины к разным браузерам, например,
для Firefox можно использовать Greasemonkey (https://addons.mozilla.org/ru/firefox/addon/greasemonkey/),
а для Safari - Tampermonkey (http://tampermonkey.net/index.php?ext=dhdg&browser=safari).
